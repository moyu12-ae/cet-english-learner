# 四六级英语深度学习 Skill

基于费曼学习法，以"做"和"用"驱动"学"。不是孤立背单词，而是在真实语境中学习词汇和表达。

支持**英语文章**和**视频字幕**两种语料，自动识别并适配处理策略。

## 功能

- 语料自动识别（文章/字幕模式）
- 四六级词汇提取与分级（CET4/CET6/超纲推荐）
- 关键句式分析与适用场景说明
- 逐段精读：词汇拆解、句式分析、流畅翻译、文化背景注
- 费曼复述引导：理解检测问题、段落复述、翻译还原
- 墨墨记忆卡导入文件生成（词汇挖空卡/句子背记卡/多选题/场景还原卡）
- 结构化文件输出

## 输出结构

每次学习完成后，自动生成以下目录：

```
{title-slug}-learning/
├── source.txt       # 原始语料副本
├── 词汇表.csv       # 四六级词汇表（CSV，可导入 Excel/Notion）
├── 关键句式.md       # 段落概览 + 关键句式分析
├── 精读笔记.md       # 逐段拆解 + 翻译 + 文化注
├── 费曼练习.md       # 理解检测问题 + 复述任务 + 翻译还原
└── 墨墨卡片.txt      # 墨墨记忆卡标准导入格式
```

## 使用方式

在 Claude Code 中提供英语文本，说：

- "帮我精读这篇文章"
- "分析这篇英语新闻"
- "帮我从这个字幕学英语"
- "生成墨墨记忆卡卡片"

Skill 将自动触发，按六阶段工作流执行。

## 可配置项

| 配置项 | 选项 | 默认值 |
|--------|------|--------|
| 目标级别 | CET4 / CET6 / 两者 | 两者 |
| 挖空策略 | 宽松 / 标准 / 密集 | 标准 |
| 超纲词处理 | 忽略 / 标记不挖空 / 一并挖空 | 忽略 |
| 句子选择偏好 | 长难句优先 / 实用短句优先 / 平衡 | 实用短句优先 |
| 费曼复述深度 | 仅问题 / 问题+复述 / 完整深度 | 仅问题 |
